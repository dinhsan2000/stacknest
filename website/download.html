Installation Guides

Windows Installation ▼

1. Download the installer (.exe file)

2. Run the installer and follow the setup wizard

3. Choose installation directory (default: Program Files)

4. Wait for installation to complete

5. Launch Stacknest from Start menu or desktop shortcut

macOS Installation ▼

1. Download the DMG file

2. Double-click to mount the DMG

3. Drag Stacknest to Applications folder

4. Eject the DMG

5. Launch from Applications folder

Note: Allow permissions in System Preferences if prompted

Linux Installation ▼

AppImage: Make executable, run directly

chmod +x stacknest-*.AppImage

.deb: Install via package manager

sudo apt install ./stacknest*.deb
